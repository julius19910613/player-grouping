# 聊天助手 - 用户使用指南

## 概述

聊天助手是一个智能对话工具，可以帮助你管理球员、执行分组、查询统计等。它使用 AI 技术，支持自然语言交互和快捷命令。

## 快速开始

### 1. 打开聊天助手

- **方式 1**：点击导航栏中的 "智能助手" 标签
- **方式 2**：使用快捷键 `Cmd/Ctrl + K` 快速打开

### 2. 发送消息

- **方式 1**：在输入框输入消息，点击 "发送" 按钮
- **方式 2**：在输入框输入消息，按 `Enter` 键发送
- **换行**：按 `Shift + Enter` 换行

### 3. 使用快捷命令

输入 `/` 查看所有快捷命令，包括：

- `/查看所有球员` - 显示所有球员列表
- `/随机分组` - 快速创建随机分组
- `/最佳阵容` - 查看推荐的最佳阵容
- `/能力分析` - 分析球员能力分布
- `/搜索球员` - 搜索特定球员
- `/查看统计` - 查看整体统计数据

## 功能详解

### 1. 球员查询

**示例对话：**
```
用户：显示所有后卫
助手：找到了 5 名后卫：
1. 张三 (总体能力: 85)
2. 李四 (总体能力: 88)
...
```

**支持的筛选条件：**
- 按位置：后卫、前锋、中锋
- 按能力：最低能力值
- 按姓名：模糊搜索

### 2. 智能分组

**示例对话：**
```
用户：把所有球员分成 2 个队
助手：已创建 2 个队伍：
- Team 1：张三, 李四, 王五 (总战力: 263)
- Team 2：赵六, 钱七, 孙八 (总战力: 260)
平衡度: 98%
```

**支持策略：**
- **平衡分组**：根据能力值均匀分配
- **随机分组**：完全随机分配

### 3. 球员分析

**示例对话：**
```
用户：分析张三的能力
助手：张三的能力分析：
- 投篮：92 ⭐⭐⭐⭐⭐
- 组织：85 ⭐⭐⭐⭐
- 防守：78 ⭐⭐⭐
- 篮板：70 ⭐⭐
- 身体素质：88 ⭐⭐⭐⭐
总体能力：85
```

### 4. 联网搜索

**示例对话：**
```
用户：搜索 NBA 最新新闻
助手：找到了 5 条相关新闻：
1. 【NBA官网】湖人击败勇士，詹姆斯砍下三双
   https://nba.com/...
   发布时间：2024-03-15
   
2. ...
```

## 高级功能

### 1. 消息操作

- **复制消息**：将鼠标悬停在消息上，点击复制图标
- **重新生成**：将鼠标悬停在 AI 回复上，点击重新生成图标

### 2. 对话管理

- **清空对话**：点击右上角 "清空对话" 按钮
- **历史记录**：对话历史会自动保存

### 3. 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `Cmd/Ctrl + K` | 打开聊天/聚焦输入框 |
| `/` | 显示快捷命令 |
| `Enter` | 发送消息 |
| `Shift + Enter` | 换行 |
| `Escape` | 关闭自动补全 |

## 常见问题

### Q: 为什么 AI 有时回复很慢？
A: AI 需要处理你的请求，可能涉及：
- 查询数据库
- 执行分组算法
- 联网搜索

### Q: 为什么提示"请求超时"？
A: 可能的原因：
- 网络连接不稳定
- 请求过于复杂
- 服务器繁忙

**解决方案：**
1. 尝试简化问题
2. 使用快捷命令
3. 稍后重试

### Q: 如何重新生成 AI 回复？
A: 将鼠标悬停在 AI 消息上，点击重新生成图标（↻）

### Q: 对话记录会保存多久？
A: 对话记录保存在本地，清除浏览器数据后会丢失。

## 最佳实践

### 1. 使用快捷命令

快捷命令更快速、更准确，推荐用于常用操作。

### 2. 明确表达需求

**推荐：**
```
把能力值 80 以上的后卫分成 2 个队
```

**不推荐：**
```
分组
```

### 3. 利用上下文

AI 会记住对话上下文，可以分步完成任务：

```
用户：显示所有后卫
助手：找到 5 名后卫...

用户：把其中能力最高的 4 个分成 2 个队
助手：已创建分组...
```
