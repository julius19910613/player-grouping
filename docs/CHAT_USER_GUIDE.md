# 智能助手使用指南

## 概述

智能助手是一个集成在球员分组工具中的 AI 助手，可以帮助你查询球员信息、执行分组操作、分析数据等。

## 功能特性

### 🚀 性能优化

- **虚拟滚动**：当消息超过 50 条时，自动启用虚拟滚动，确保流畅的用户体验
- **Markdown 渲染**：支持代码高亮、列表、表格等丰富的格式
- **智能防抖**：输入自动防抖，减少不必要的渲染

### ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Cmd/Ctrl + K` | 聚焦聊天输入框 |
| `Enter` | 发送消息 |
| `Shift + Enter` | 换行 |
| `/` | 显示快捷命令 |
| `↑` / `↓` | 导航快捷命令 |
| `Tab` / `Enter` | 选择快捷命令 |
| `Esc` | 关闭自动补全 |

### 💬 消息操作

#### 复制消息
- 鼠标悬停在消息上，点击复制按钮 📋
- 支持复制用户和助手消息

#### 重新生成
- 仅助手消息支持重新生成
- 点击重新生成按钮 🔄 重新获取回复

#### 清空对话
- 点击顶部"清空对话"按钮
- 确认后清除所有消息历史

### 🎯 快捷命令

输入 `/` 查看可用的快捷命令：

- `/查看所有球员` - 显示所有球员列表
- `/随机分组` - 执行随机分组
- `/最佳阵容` - 生成最佳阵容
- `/能力分析` - 分析球员能力
- `/搜索球员` - 搜索特定球员
- `/查看统计` - 查看统计数据

### ♿ 无障碍支持

- **键盘导航**：完全支持键盘操作
- **屏幕阅读器**：所有元素都有 ARIA 标签
- **实时公告**：消息状态变化时有语音提示

## 使用示例

### 示例 1：查询球员

```
用户：查看所有前锋位置的球员
助手：正在查询前锋位置的球员...
```

### 示例 2：执行分组

```
用户：帮我把这 10 个球员分成两队
助手：好的，我将根据球员能力进行平衡分组...
```

### 示例 3：数据分析

```

助手：根据数据分析，球员 A 在投篮方面更强...
```

## 常见问题

### Q: 消息发送失败怎么办？
A: 
1. 检查网络连接
2. 点击错误提示中的"重试"按钮
3. 刷新页面重试

### Q: 如何查看历史消息？
A: 目前暂不支持历史消息加载，每次打开都是新会话。后续版本会支持。

### Q: 助手支持哪些功能？
A: 助手可以：
- 查询球员信息
- 执行分组操作
- 分析球员数据
- 回答篮球相关问题
- 执行搜索操作

### Q: 消息有字数限制吗？
A: 单条消息最多 1000 字符。

## 技术细节

### 性能优化

1. **虚拟滚动**
   - 使用 `react-window` 实现
   - 超过 50 条消息自动启用
   - 消息高度动态计算

2. **Markdown 渲染**
   - 使用 `react-markdown`
   - 支持 GFM（GitHub Flavored Markdown）
   - 代码高亮（highlight.js）
   - 组件缓存优化

3. **防抖策略**
   - 输入防抖：300ms
   - 自动补全防抖：300ms

### 无障碍特性

1. **ARIA 标签**
   - `role="log"` 消息列表
   - `role="article"` 消息项
   - `aria-live="polite"` 实时更新

2. **键盘支持**
   - Tab 键导航
   - Enter 键确认
   - Esc 键取消

3. **屏幕阅读器**
   - 状态变化公告
   - 错误提示朗读
   - 操作反馈
